--- 3c. Root Partition UUID ---

How GRUB uses UUID: GRUB passes the UUID to the kernel via the 'root=' parameter.
The kernel then uses this UUID to locate and mount the correct root partition,
even if device names (like /dev/sda1) change between boots.
